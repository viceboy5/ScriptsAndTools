Operators

Arithmetic 

+ : Addition 3.5 + 5.2 => 8.7
            ("Hello " + "World") = "Hello World"

- : Subtraction 7.3 - 2.1 => 5.2

* : Multiplication   3 * 5 => 15

/ : Division    15 / 3 => 5

++ : Increment    x + 1
                int $num = 13;
                $num++;
                $num = $num + 1

-- : Decrement x + 1
                int $num = 13;
                $num--;  //num is now 12
                $num = $num - 1

% : Modulus     13 % 2 => 3

^ : Power       3 ^ 5
                3 * 3 * 3 * 3 * 3
